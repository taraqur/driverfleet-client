# DriveFleet

DriveFleet is a premium car rental platform offering high-end automotive experiences and luxury rentals. Discover and book your dream car with ease through our modern, fast, and secure web application.

**Live Site URL:** https://client-iota-henna.vercel.app/

## Features

- **Premium Vehicle Selection:** Browse through a curated collection of high-end, luxury, and exotic cars available for rent.
- **Secure Authentication:** Robust user authentication and authorization utilizing Better-Auth and Firebase for a seamless login experience.
- **Modern & Responsive UI:** Designed with Tailwind CSS and DaisyUI, ensuring a stunning and fully responsive experience across all devices.
- **Smooth Animations:** Integrated Framer Motion to provide engaging micro-interactions and smooth page transitions.
- **Dynamic Theming:** Built-in support for Light and Dark modes, allowing users to customize their browsing experience.
